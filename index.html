關於這個網站

這是一個「不用後端也能有互動」的基礎網站：使用 localStorage 記錄主題與待辦事項， 並提供簡潔的排版結構，適合當作專題作品集或活動介紹頁的起點。

易修改

整體只分成 HTML / CSS / JS 三檔，結構直覺。

可擴充

你可以加上更多區塊、圖片輪播、甚至接 API。

有互動

主題切換、待辦、瀏覽計數、時間顯示。
